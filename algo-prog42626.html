[JAVA/프로그래머스] 힙(Heap)_더 맵게

HEESOO 04 Nov 2019 on Programmers, Heap, Priorityqueue

문제

https://programmers.co.kr/learn/courses/30/lessons/42626

( ᐛ )و 첫 번째 도전

1. 설계

  1. 가장 맵지 않은, scoville이 낮은 원소를 사용하므로 우선순위가 낮은 순서로 정렬, 리턴하는 우선순위큐를 사용한다.
  2. 하나라도 K보다 작다면, 주어진 공식을 이용해 새 조합을 만든다.

2. 구현

import java.util.PriorityQueue;
/**
 *
 * @author HEESOO
 *
 */
 class Solution {
     public int solution(int[] scoville, int K) {
         int answer = 0;
         int a,b;
         PriorityQueue<Integer> pq=new PriorityQueue<>();
         for(int i:scoville){
             pq.offer(i);
         }
         while(pq.peek()<K){
             if(pq.size()==1){
                 return -1;
             }
             else{
                 a=pq.poll();
                 b=pq.poll();
                 pq.offer(a+(b*2));
                 answer++;
             }
         }
         return answer;
     }
 }
  • PriorityQueue pq: 스코빌 지수가 낮은 순으로 뽑을 수 있게 우선순위큐를 사용한다.
