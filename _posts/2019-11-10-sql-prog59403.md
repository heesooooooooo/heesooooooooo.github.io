### 1. 설계
1. 동물의 아이디와 이름을 출력하므로 `SELECT ANIMAL_ID, NAME`이다.
2. ANIMAL_ID 순으로 정렬하므로 ORDER BY를 사용한다.

### 2. 구현
```sql
SELECT ANIMAL_ID, NAME FROM ANIMAL_INS ORDER BY ANIMAL_ID;
```
