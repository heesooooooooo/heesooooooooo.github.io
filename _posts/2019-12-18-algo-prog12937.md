### 1. 설계
1. 나머지 연산으로 짝수일 경우 Even, 홀수면 Odd를 리턴한다.

### 2. 구현 (성공 코드)
```java
/**
 *
 * @author HEESOO
 *
 */
 class Solution {
   public String solution(int num) {
       String answer = "";
       return answer=num%2==0? "Even":"Odd";
   }
 }
 ```
