## <span style="color:navy">( ᐛ )و 도전</span>

### 1. 설계
1. 팩토리얼을 계산한다.

### 2. 구현 (성공 코드)
```java
import java.util.Scanner;

/**
 * 
 * @author HEESOO
 *
 */
public class Main {
	public static void main(String[] args) {
		Scanner input=new Scanner(System.in);
		int n=input.nextInt();
		int answer=n;
		if(n==0) answer=1;
		for(int i=n-1;i>0;i--){
			answer*=i;
		}
		System.out.println(answer);
	}
}

 ```
