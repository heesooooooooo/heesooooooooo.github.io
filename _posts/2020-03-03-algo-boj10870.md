### 1. 설계

### 2. 구현 (성공 코드)
```java
import java.util.Scanner;

/**
 * 
 * @author HEESOO
 *
 */
public class Main {
	public static int fib(int n){
		if(n==0) return 0;
		else if(n==1) return 1;
		else return fib(n-1)+fib(n-2);		
	}
	public static void main(String[] args) {
		Scanner input=new Scanner(System.in);
		int n=input.nextInt();
		System.out.println(fib(n));
		
	}
	
}

 ```
