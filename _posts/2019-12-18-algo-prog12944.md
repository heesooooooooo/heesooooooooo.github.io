### 1. 설계
1. for-each로 배열의 원소들을 모두 더한 뒤, 배열 길이만큼 나눠서 평균값을 리턴한다.

### 2. 구현 (성공 코드)
```java
/**
 *
 * @author HEESOO
 *
 */
 class Solution {
   public double solution(int[] arr) {
       double answer = 0;
       for(int num:arr){
           answer+=num;
       }
       answer/=arr.length;
       return answer;
   }
 }
 ```
