## <span style="color:navy">문제</span>
<https://www.acmicpc.net/problem/2309>

## <span style="color:navy">( ᐛ )و 도전</span>

### 1. 설계
1. 오름차순으로 출력해야하므로 정렬한다.
2. 난쟁이 키의 합이 100이 되도록 제거해야할 두 개의 값을 이중for문을 통해 찾는다.

### 2. 구현 (성공 코드)
```java
import java.util.Arrays;
import java.util.Scanner;

/**
 * @author HEESOO
 *
 */
public class Main {
	public static void print(int[] array, int x, int y) {
		for(int i=0;i<9;i++) {
			if(i!=x&&i!=y)
				System.out.println(array[i]);
		}
	}
	public static void main(String[] args) {
		// TODO Auto-generated method stub
		Scanner scan = new Scanner(System.in);
		int[] array=new int[9];
		int sum=0;
		for(int i=0;i<9;i++) {
			array[i]=scan.nextInt();
			sum+=array[i];
		}
		Arrays.sort(array);
		
		for(int i=0;i<9;i++) {
			for(int j=i+1;j<9;j++) {
				if(sum-array[i]-array[j]==100) {
					print(array,i,j);
					return;
				}
			}
		}
		
	}
}

 ```

- 출력을 위해 오름차순으로 정렬한다.
- 이중for문을 이용해 i, j를 선택하고 그 두 개를 뺐을 때 100이 된다면 print()메소드를 호출해 두 개를 제외한 나머지들을 출력한 후 return으로 종료한다.
