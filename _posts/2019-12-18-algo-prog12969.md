### 1. 설계
1. 가로 길이, 세로 길이에 맞는 직사각형을 출력한다.

### 2. 구현 (성공 코드)
```java
/**
 *
 * @author HEESOO
 *
 */
 import java.util.Scanner;

 public class Solution {
     public static void main(String[] args) {
         Scanner sc = new Scanner(System.in);
         int a = sc.nextInt();
         int b = sc.nextInt();

         //System.out.println(a + b);

         for(int i=0;i<b;i++){
             for(int j=0;j<a;j++){
                 System.out.print("*");
             }
             System.out.println();
         }
     }
 }
 ```
