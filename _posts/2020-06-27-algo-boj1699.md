		int[] dp=new int[n+1];
		dp[1]=1;
		for(int i=2;i<=n;i++) {
			dp[i]=i;
			for(int j=1;j*j<=i;j++) {
				dp[i]=Math.min(dp[i], dp[i-j*j]+1);
			}
		}

	}

### 4. 설명
1. **<span style="color:navy">DP 규칙을 찾아낸다</span>**  

| i | dp[i] | ㅁ |  
| :---: | :---: | :---: |  
| 1 | 1 | 1 |  
| 2 | 2 | 1+1 |  
| 3 | 3 | 1+1+1 |  
| 4 | 1 | 2^2 |  
| 5 | 2 | 2^2+1 |  
| 6 | 3 | 2^2+1+1 |  
| 7 | 4 | 2^2+1+1+1 |  
| 8 | 1 | 2^3 |  
| 9 | 2 | 2^3+1 |  
| 10 | 3 | 2^3+1+1 |  
| 11 | 4 | 2^3+1+1+1 |  
| 12 | 5 | 2^3+1+1+1+1 |   

- 자연수 i의 제곱수 합을 찾기 위해 j를 이용하여 1부터 j까지의 제곱수를 이용하여 값을 구한다.
