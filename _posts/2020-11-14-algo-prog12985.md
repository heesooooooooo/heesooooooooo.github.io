## <span style="color:navy">👊 도전</span>

### 1. 설계
1. x가 짝수라면 다음 라운드에서 x는 x/2 번째가 된다.
2. 홀수라면 x/2+1가 된다.
3. a, b가 같아질 때까지 진행한다.


### 2. 구현 
{% highlight java linenos %}
class Solution
{
    public int solution(int n, int a, int b)
    {
        int answer=0;
        while(a!=b){
            a=solve(a);
            b=solve(b);
            answer++;
        }

        return answer;
    }
    
    public int solve(int x){
        if(x%2==0) return x/2;
        else return x/2+1;
    }
}
{% endhighlight %}
