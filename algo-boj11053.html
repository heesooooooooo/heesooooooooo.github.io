[JAVA/백준] 동적 계획법 1: 가장 긴 증가하는 부분 수열

HEESOO 20 Mar 2020 on Boj, Dp

문제

https://www.acmicpc.net/problem/11053

( ᐛ )و 도전

1. 설계

  1. dp[i]=array[i]를 가장 큰 원소로 하는 가장 긴 증가하는 부분의 수열의 길이로 한다.

2. 구현 (성공 코드)

import java.util.Arrays;
import java.util.Scanner;

/**
 * 
 * @author HEESOO
 *
 */
public class Main {
	public static void main(String[] args){
		Scanner input=new Scanner(System.in);
		int n=input.nextInt();
		int[] array=new int[n];
		int answer=0;
		for(int i=0;i<n;i++){
			array[i]=input.nextInt();
		}

		int[] dp=new int[n];
		Arrays.fill(dp, 1);
		
		for(int i=1;i<n;i++){
			for(int j=0;j<i;j++){
				if(array[i]>array[j]&&dp[j]>=dp[i]){
					dp[i]=dp[j]+1;
				}
			}
		}
		
		Arrays.sort(dp);
		System.out.println(dp[n-1]);
	}
}

3. 결과

성공٩(˘◊˘)۶

4. 설명

  1. DP를 이용한다.
    • i를 기준으로 j로 0부터 i까지 확인하며, i가 제일 큰 값이 될 수 있도록 i보다 작은 j를 찾고, dp[j]가 dp[i]보다 같거나 크다면, dp[i]는 dp[j]+1의 값을 가진다.
    • Arrays.sort()로 정렬하여 가장 큰 값을 리턴한다.
