Comparable과 Comparator 차이

1. Comparable

  • 오름차순, 가나다 등 기본 정렬 기준을 따르는데, 새 클레스에서 정렬할 기준(변수)을 잡아줄 때 사용
  • 기존 Integer, String 등의 compareTo()를 이용하는게 보통
  • compareTo()를 오버라이드해서 사용, 파라미터는 하나
  • 나 자신과 파라미터를 비교

2. Comparator

  • 내림차순, 사전 역순 등 기존 정렬 기준이 아닌 새로운 것을 짤 때 사용
  • compare()을 오버라이드
